um belo exemplo de navegação do DOM e o seguinte

let teste = getElementsbyTagName("idmain")
teste.getElementsbyTagName("p")

O código primeiro pega o elemento `<main>` pelo ID `"idmain"`, depois busca todos os `<p>` dentro dele. 
Como os `<p>` estão dentro do `<main>`, eles são considerados seus filhos.

quando for usar o "querySelector" para manipular o DOM ela só irar buscar um objeto, ou seja, ela ira pegar o primeiro objeto.  

Para manipular o DOM precisar ter certeza que o HTML tenha cido totalmente lido no Browser.

Para mosttrar todos os elementos filhos em JS e so usar o elemento "children" Já o elemento psi e só utilizar o "parentElement"

O elemento "remove()" remove o elemento do DOM 

O método "insertBefore()" é uma função da API do DOM usada para inserir um nó (elemento) antes de outro nó filho dentro de um elemento pai.
parentNode.insertBefore(novoElemento, elementoDeReferencia);
